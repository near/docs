## 前提条件

开始之前，请确保设置好您的开发环境！ 

<Accordion title="在 Windows 上工作？">

  请参阅我们的博客文章，获取在 Windows 上使用 NEAR 入门的逐步指南，了解如何设置 WSL 和您的环境。

</Accordion>

<Tabs>
  <Tab title="🌐 JavaScript">

    ```bash
    # 使用 nvm 安装 Node.js（更多选项请参见：https://nodejs.org/en/download）
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
    nvm install latest

    # 安装 NEAR CLI 以部署和与合约交互
    curl --proto '=https' --tlsv1.2 -LsSf https://github.com/near/near-cli-rs/releases/latest/download/near-cli-rs-installer.sh | sh
    ```

  </Tab>

  <Tab title="🦀 Rust">

    ```bash
    # 安装 Rust：https://www.rust-lang.org/tools/install
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

    # 合约将被编译为 wasm，所以我们需要添加 wasm 目标
    rustup target add wasm32-unknown-unknown

    # 安装 NEAR CLI 以部署和与合约交互
    curl --proto '=https' --tlsv1.2 -LsSf https://github.com/near/near-cli-rs/releases/latest/download/near-cli-rs-installer.sh | sh

    # 安装 cargo near 以帮助构建合约
    curl --proto '=https' --tlsv1.2 -LsSf https://github.com/near/cargo-near/releases/latest/download/cargo-near-installer.sh | sh
    ```

  </Tab>

</Tabs>

我们将使用 [NEAR CLI](/tools/cli) 通过终端与区块链交互，您可以选择 JavaScript 或 Rust 来编写合约。
