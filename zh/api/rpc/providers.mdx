## 发起请求

所有端点均接受 JSON-RPC 2.0 `POST` 请求。NEAR RPC 使用非标准格式，方法名称编码在 URL 路径中而非请求体中。

```bash
curl -X POST https://rpc.mainnet.near.org/status \
  -H "Content-Type: application/json" \
  -d '{}'
```
