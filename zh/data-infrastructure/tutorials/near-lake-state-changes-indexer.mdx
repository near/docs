本教程将引导您使用 NEAR Lake Framework 构建一个简单的索引器。该索引器将监听 `StateChange` 事件并打印有关账户变更的相关数据。

**教程源代码**

</Note>

同时，我们持续更新源代码以与已发布 crate 的最新版本保持同步。

</Info>

我们创建了一个视频教程，以庆祝 [NEAR Lake Framework](/data-infrastructure/near-lake-framework) 的发布公告。

在本教程中，您将构建一个索引器应用，用于监控影响所提供账户或账户列表的任何 `StateChange`。
