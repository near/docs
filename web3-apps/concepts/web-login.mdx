NEAR offers multiple options to connect users via their wallets. The recommended approach is [NEAR Connect](../tutorials/wallet-login), a zero-dependency library that runs wallet code in isolated sandboxed iframes for maximum security.

Once the user is logged in, they can interact with the NEAR blockchain — calling smart contracts, transferring tokens, and more.

---

## NEAR Connect

[NEAR Connect](../tutorials/wallet-login) is a zero-dependency lightweight library that allows users to connect to your dApp using their preferred wallet. It uses a secure sandbox-based architecture where wallet scripts run in isolated iframes.

![Preview](https://github.com/user-attachments/assets/c4422057-38bb-4cd9-8bd0-568e29f46280)

<Tip>
Follow the step-by-step [Wallet Login tutorial](../tutorials/wallet-login) to integrate NEAR Connect into your app.
</Tip>

**Key features:**
- **Secure Execution**: Wallet scripts run in isolated sandboxed iframes
- **Dynamic Wallets**: Wallets update without requiring code changes
- **Zero Dependencies**: Lightweight with no external dependencies
- **Wide Support**: HOT Wallet, Meteor, Nightly, WalletConnect, and 10+ others
